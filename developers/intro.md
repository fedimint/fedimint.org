# Build on Fedimint

Fedimint is an open-source, modular protocol — and we want developers to build on it. Whether you're building a wallet, integrating eCash payments into an existing app, or exploring the module system, the SDK documentation has everything you need to get started.

**[Go to the Fedimint SDK documentation](https://sdk.fedimint.org)**
